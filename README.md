# React Assembly Line

**React Assembly Line** is an interactive, gamified learning experience designed to teach you React concepts. Progress through 25 levels by writing real React code to assemble a high-tech humanoid robot!

## 🎮 Play the Game
[**Play React Assembly Line Live!**](https://chakanevaibhav.github.io/react-learn-game/)

*(Note: The game requires you to fill in the correct React syntax to proceed to the next level).*

## 🚀 Features
- **25 Gamified Levels**: Master everything from basic JSX to Advanced State Management, Hooks (`useState`, `useEffect`, `useContext`, `useRef`), and API integration.
- **Interactive Code Editor**: Fill in the missing React code blocks to solve each directive. The input is strictly validated!
- **Visual Progression System**: Watch a futuristic Tesla Optimus-style humanoid robot assemble itself piece-by-piece as you complete each level.
- **Sci-Fi HUD Overlays**: Unlock high-tech visual achievements (laser targeting, optical scanners, companion drones, uplink signals) that map directly to your coding milestones.
- **Locked Progression Constraints**: You must successfully complete your current directive to unlock the next level, ensuring a solid foundation of React knowledge.

## 🛠️ Technology Stack
- **React 18** (via Vite)
- **TypeScript**
- **Vanilla CSS** (Custom Neon Sci-Fi aesthetics)
- **Lucide Icons**

## 💻 Local Development
To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/chakanevaibhav/react-learn-game.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd react-learn-game
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open your browser to the local port displayed in the console (usually `http://localhost:5173/` or `http://localhost:5174/`).

## 🤖 Levels & Curriculum Structure
The curriculum is designed to take you from a complete beginner to writing advanced React patterns:
- **Levels 01-05:** JSX syntax, components, rendering logic, and passing props.
- **Levels 06-10:** Event handling, conditionals, mapping arrays, and keys.
- **Levels 11-15:** State management (`useState`), controlled forms, and the component lifecycle (`useEffect`).
- **Levels 16-20:** Advanced Hooks and performance (`useMemo`, `useCallback`, `useRef`, `useReducer`).
- **Levels 21-25:** Global state sharing (Context API), Custom Hooks, Async API fetching, and React Portals.
